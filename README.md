# Deepak Kumar — Portfolio

A professional single-page portfolio built with React (Vite), showcasing career trajectory, skills, education, publication, and Medium blogs.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build for production

```bash
npm run build
```

Output is in the `dist/` folder.

## Sections

- **Hero** — Name, tagline, GitHub & Medium links  
- **About** — Professional summary  
- **Experience** — Timeline: CARS24, Thoughtworks, Nagarro  
- **Skills** — Tech stack tags  
- **Education** — B.Tech & Diploma  
- **Publication** — ICACSE 2019 paper  
- **Blogs** — Medium articles with links  
- **Contact** — Email, phone, location, GitHub, Medium  

Content is driven by `src/data/content.js`; update that file to change copy, links, or add new posts.
